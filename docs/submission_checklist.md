## Report Rules

- Use required outer cover and title page templates.
- Include signed declaration on title page.
- Main body limit: **max 15,000 words** and **max 40 A4 pages**.
- Font guideline: **Calibri 12**.
- Suggested structure:
  - Introduction
  - Motivation
  - Related Work
  - Description of Work
  - Methodology
  - Design
  - Implementation
  - Evaluation
  - Summary and Reflections
  - Project Management
  - Contributions and Reflections
  - Bibliography
  - Optional Appendices

## Demo Video Rules

- Duration: **1 to 3 minutes**
- Format: **MP4**
- Encoding: **H264**
- Resolution: **1280x720**
- First frame must include:
  - student ID
  - project title
- Focus on project outcome and your contribution.
